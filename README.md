SecCMD (Security Command)
===

`SecCMD` 安全命令【速查手册】，为安全人员提供快速参考备忘清单，更快速获取安全人员所需的各种各样的安全操作命令。快一步响应，多一倍安全。希望 `SecCMD` 能够帮助安全人员节省命令查找时间、提升安全操作效率，帮助企业公司降低安全损失。
